# AI Trading Bot v3.5

## Projektstruktur

```
ai_bot/
├── config/              # Konfiguration
├── src/                 # Quellcode
│   ├── ml_integration/  # ML-Komponenten
│   ├── strategies/      # Strategien
│   ├── self_improvement/# Selbstoptimierung
│   ├── data/            # Datenverarbeitung
│   ├── trading/         # Trading Engine
│   └── utils/           # Hilfsfunktionen
├── models/              # ML-Modelle
├── data/                # Daten & Performance
├── logs/                # Logs
├── tests/               # Tests
├── docs/                # Dokumentation
├── scripts/             # Start-Skripte
├── experiments/         # Experimente
├── archive/             # Archiv
├── main.py              # Hauptskript
└── requirements.txt     # Abhängigkeiten
```

## Installation

```bash
pip install -r requirements.txt
python main.py
```

## Verwendung

1. Konfiguration anpassen: `config/bot_config.yaml`
2. Bot starten: `python main.py`
3. Paper Trading Modus wählen (Option 1)
