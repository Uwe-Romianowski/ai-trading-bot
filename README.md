# AI Forex Trading Bot v4.0

Forex trading bot with Machine Learning and MetaTrader 5 integration.

## Features
- ML Model: Random Forest with 41 features
- Live MT5 data from Demo Account
- Confidence-based trading signals
- Risk management with stop-loss

## Quick Start
```bash
git clone https://github.com/Uwe-Romianowski/ai-trading-bot.git
cd ai-trading-bot
pip install -r requirements.txt
python main.py
